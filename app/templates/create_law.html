Создать новый закон

Напишите Python код, который будет определять логику работы вашего закона

law_code.py
Код должен содержать функцию apply(), которая возвращает результат работы закона
Укажите через запятую действия, которые должны запускать этот закон
Если отключено, закон потребует голосования для активации

API для законов

api.get_user(user_id?)
Получить данные пользователя
api.get_party(party_id)
Получить данные партии
api.count_users_where(**conditions)
Подсчитать пользователей по условиям
api.get_government()
Получить данные правительства
api.log_action(action, details)
Записать действие в лог
context.user_id
ID пользователя, который вызвал закон
context.action
Действие, которое вызвало закон
context.data
Дополнительные данные о действии

Примеры законов

  • Закон о голосовании
  • Минимальный размер партии
  • Ограничение срока правления
  • Ежедневная проверка
